使用示例

1. 使用 curl（无限制输出）

curl -X POST "https://glm.study-llm.me/chat" \ -H "Content-Type: application/json" \ -d '{ "prompt": "请详细解释人工智能的发展历史", "max_tokens": null, "temperature": 0.7, "stream": false, "full_response": true }'

2. 使用 JavaScript Fetch（无限制输出）

// 无限制输出的 API 调用 fetch('https://glm.study-llm.me/chat', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ prompt: '请详细解释量子力学的基本概念', max_tokens: null, // 无令牌限制 temperature: 0.7, // 平衡的创造力 stream: false, // 一次性获取完整响应 full_response: true // 请求完整响应 }) }) .then(response => response.json()) .then(data => console.log(data));
