🤖 GLM Worker API 测试

📡 API 端点

GET /health - 健康检查
POST /chat - 聊天对话

🔍 健康检查测试

💬 聊天测试

📋 使用示例

1. 健康检查 (curl)

curl -X GET "http://localhost:8787/health" \ -H "Content-Type: application/json"

2. 简单提示 (curl)

curl -X POST "http://localhost:8787/chat" \ -H "Content-Type: application/json" \ -d '{ "prompt": "你好，请介绍一下数学的基本概念" }'

3. 消息数组格式 (curl)

curl -X POST "http://localhost:8787/chat" \ -H "Content-Type: application/json" \ -d '{ "messages": [ { "role": "user", "content": "你好，请介绍一下数学的基本概念" } ] }'

4. JavaScript 调用示例

// 健康检查 fetch('/health') .then(response => response.json()) .then(data => console.log(data)); // 聊天对话 fetch('/chat', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ messages: [ { role: 'user', content: '你好，请介绍一下数学的基本概念' } ] }) }) .then(response => response.json()) .then(data => console.log(data));
