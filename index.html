关于学习助手

这是一个由好奇心、探索欲与热忱促成的小小实验项目，它始于几个萦绕心头的问题：

  • 生成式AI如此强大，如何让它适配个性化需求？
  • 在AI时代，编程与创造是否真的人人可为？
  • 如何利用AI帮助孩子学习？
以下是实践过程中的一些感悟：
  • 拒绝躺平，开启思考
  • 不要拖延，立即行动
  • 无惧试错，在实践中成长
  • 永不言弃，探索不止
  • 坚信自己，终有所成

  • 音频生成：MINIMAX API
  • 编译器：Cursor
  • 代码生成：Claude-3.7-sonnet
  • 网页部署：Cloudflare + Github
