API Usage Examples

Simple Prompt Format

POST /api/glm Content-Type: application/json { "prompt": "Hello, can you introduce yourself?" }

Messages Array Format

POST /api/glm Content-Type: application/json { "messages": [ {"role": "system", "content": "You are a helpful assistant"}, {"role": "user", "content": "Hello, how are you?"} ] }

JavaScript Example

async function callGlmApi(prompt) { const response = await fetch('/api/glm', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ prompt: prompt }) }); const data = await response.json(); return data.output; }
