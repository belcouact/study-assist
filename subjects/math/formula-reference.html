数学公式参考

中学数学核心公式速查手册，涵盖代数、几何、三角函数、统计等重要内容

代数公式

因式分解

$a^2 - b^2 = (a+b)(a-b)$
平方差公式
两个数的平方差等于这两个数的和与这两个数的差的乘积。这是代数中最基本且常用的因式分解公式之一。
例题：
分解因式：$x^2 - 9$
解：$x^2 - 9 = x^2 - 3^2 = (x+3)(x-3)$
$(a \pm b)^2 = a^2 \pm 2ab + b^2$
完全平方公式
两数和（或差）的平方，等于两数的平方和，加上（或减去）两数乘积的两倍。这个公式用于展开平方式或进行完全平方分解。
例题：
展开：$(x+3)^2$
解：$(x+3)^2 = x^2 + 2×x×3 + 3^2 = x^2 + 6x + 9$
$(a \pm b)^3 = a^3 \pm 3a^2b + 3ab^2 \pm b^3$
完全立方公式
两数和（或差）的立方展开公式。注意符号规律：立方和中间两项符号都为正，立方差中间两项符号一正一负。
例题：
展开：$(x-2)^3$
解：$(x-2)^3 = x^3 - 3x^2×2 + 3x×2^2 - 2^3 = x^3 - 6x^2 + 12x - 8$
$a^3 \pm b^3 = (a \pm b)(a^2 \mp ab + b^2)$
立方和差公式
立方和或立方差的因式分解公式。立方和分解为(a+b)乘以不完全平方差，立方差分解为(a-b)乘以不完全平方和。
例题：
分解因式：$x^3 + 8$
解：$x^3 + 8 = x^3 + 2^3 = (x+2)(x^2-2x+4)$

二次方程

$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$
二次方程求根公式
对于一般式二次方程$ax^2 + bx + c = 0$（a≠0），可以用此公式求解。公式中的±表示方程有两个根。
例题：
解方程：$x^2 - 5x + 6 = 0$
解：$a=1, b=-5, c=6$，$x = \frac{5 \pm \sqrt{25-24}}{2} = \frac{5 \pm 1}{2}$
所以 $x_1 = 3, x_2 = 2$
$\Delta = b^2 - 4ac$
判别式
判别式用于判断二次方程根的性质：Δ>0时有两个不等实根，Δ=0时有两个相等实根，Δ<0时无实根。
例题：
判断方程$x^2 - 4x + 5 = 0$的根的性质。
解：$\Delta = (-4)^2 - 4×1×5 = 16 - 20 = -4 < 0$
所以方程无实根

$(a^m)^n = a^{mn}$

$\log_a(xy) = \log_a x + \log_a y$
对数乘法法则
$\log_a\left(\frac{x}{y}\right) = \log_a x - \log_a y$
对数除法法则
$\log_a x^n = n\log_a x$

几何公式

平面几何

$S_{\text{三角形}} = \frac{1}{2}ah$
三角形面积
三角形面积等于底边长度与对应高的乘积的一半。这里a为底边，h为对应的高。任意一边都可以作为底边。
例题：
底边为8cm，高为6cm的三角形面积是多少？
解：$S = \frac{1}{2} × 8 × 6 = 24$ cm²
$S_{\text{三角形}} = \sqrt{s(s-a)(s-b)(s-c)}$
海伦公式 ($s = \frac{a+b+c}{2}$)
已知三角形三边长a、b、c时，可用海伦公式求面积。其中s为半周长。这个公式在只知道三边长度时特别有用。
例题：
三角形三边长为3、4、5，求面积。
解：$s = \frac{3+4+5}{2} = 6$，$S = \sqrt{6×3×2×1} = \sqrt{36} = 6$
$S_{\text{圆}} = \pi r^2$
圆的面积
圆的面积等于π乘以半径的平方。π ≈ 3.14159，这是圆周率，表示圆周长与直径的比值。
例题：
半径为5cm的圆的面积是多少？
解：$S = π × 5^2 = 25π ≈ 78.54$ cm²
$C_{\text{圆}} = 2\pi r$
圆的周长
圆的周长等于2π乘以半径，也可以写成πd（d为直径）。周长是圆的边界长度。
例题：
直径为10cm的圆的周长是多少？
解：$C = π × 10 = 10π ≈ 31.42$ cm
$S_{\text{梯形}} = \frac{(a+b)h}{2}$
梯形面积
梯形面积等于上底与下底长度之和乘以高再除以2。这里a、b为两个平行边（上底、下底），h为高。
例题：
上底4cm，下底8cm，高5cm的梯形面积是多少？
解：$S = \frac{(4+8) × 5}{2} = \frac{60}{2} = 30$ cm²

解析几何

$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
两点间距离公式
计算平面直角坐标系中两点之间距离的公式，是勾股定理在坐标系中的应用。
例题：
求点A(1,2)和点B(4,6)之间的距离。
解：$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9+16} = \sqrt{25} = 5$
$(x-a)^2 + (y-b)^2 = r^2$

基本定理

$a^2 + b^2 = c^2$
勾股定理（毕达哥拉斯定理）
直角三角形中，两条直角边的平方和等于斜边的平方。这是几何学中最重要的定理之一，广泛应用于距离计算、建筑设计、工程测量等领域。
例题：
已知直角三角形的两条直角边分别为3和4，求斜边长度。
解：$c^2 = 3^2 + 4^2 = 9 + 16 = 25$，所以 $c = 5$
$d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}$
点到直线的距离公式
点$(x_0, y_0)$到直线$ax + by + c = 0$的距离公式。这个公式在解析几何中非常重要，用于计算点到直线的最短距离。
例题：
求点$(2, 3)$到直线$3x + 4y - 5 = 0$的距离。
解：$d = \frac{|3×2 + 4×3 - 5|}{\sqrt{3^2 + 4^2}} = \frac{|6 + 12 - 5|}{\sqrt{25}} = \frac{13}{5} = 2.6$
$M(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2})$
中点坐标公式
线段中点的坐标等于两端点横坐标和纵坐标的算术平均值。常用于求线段中点和判断中点问题。
例题：
求线段AB的中点，其中A(2,3)，B(6,7)。
解：中点M$(\frac{2+6}{2}, \frac{3+7}{2}) = (4, 5)$
$k = \frac{y_2-y_1}{x_2-x_1}$
直线斜率公式
斜率表示直线的倾斜程度，等于纵坐标增量与横坐标增量的比值。斜率为正时直线上升，为负时下降。
例题：
求过点A(1,2)和B(3,8)的直线斜率。
解：$k = \frac{8-2}{3-1} = \frac{6}{2} = 3$
$d = \frac{|c_1-c_2|}{\sqrt{a^2+b^2}}$
两平行线间距离
两条平行线$ax+by+c_1=0$和$ax+by+c_2=0$之间的距离公式。只适用于同形式的平行线方程。
例题：
求$3x+4y-2=0$和$3x+4y+8=0$之间的距离。
解：$d = \frac{|-2-8|}{\sqrt{3^2+4^2}} = \frac{10}{5} = 2$

三角函数

基本关系

$\sin^2\theta + \cos^2\theta = 1$
平方和恒等式
这是三角函数中最基本的恒等式，对任意角θ都成立。常用于化简三角函数表达式和求解三角方程。
例题：
已知$\sin\theta = \frac{3}{5}$，且θ为锐角，求$\cos\theta$。
解：$\cos^2\theta = 1 - \sin^2\theta = 1 - \frac{9}{25} = \frac{16}{25}$
因为θ为锐角，所以$\cos\theta = \frac{4}{5}$
$\tan\theta = \frac{\sin\theta}{\cos\theta}$
正切定义
正切函数定义为正弦函数与余弦函数的比值。当$\cos\theta = 0$时正切函数无意义。
例题：
已知$\sin 30° = \frac{1}{2}$，$\cos 30° = \frac{\sqrt{3}}{2}$，求$\tan 30°$。
解：$\tan 30° = \frac{\sin 30°}{\cos 30°} = \frac{\frac{1}{2}}{\frac{\sqrt{3}}{2}} = \frac{1}{\sqrt{3}} = \frac{\sqrt{3}}{3}$
$1 + \tan^2\theta = \sec^2\theta$
正切-正割关系
这是三角函数的另一个重要恒等式，其中$\sec\theta = \frac{1}{\cos\theta}$是正割函数。
例题：
已知$\tan\theta = 2$，求$\sec\theta$。
解：$\sec^2\theta = 1 + \tan^2\theta = 1 + 4 = 5$
所以$\sec\theta = \pm\sqrt{5}$

$\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$

$\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$

解三角形

$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$
正弦定理
在任意三角形中，各边与其对角正弦值的比相等，且等于外接圆直径2R。常用于已知两角一边或两边一角求解三角形。
例题：
在△ABC中，已知a=8，A=60°，B=45°，求边b。
解：由正弦定理：$\frac{a}{\sin A} = \frac{b}{\sin B}$
$b = \frac{a \sin B}{\sin A} = \frac{8 \sin 45°}{\sin 60°} = \frac{8 × \frac{\sqrt{2}}{2}}{\frac{\sqrt{3}}{2}} = \frac{8\sqrt{6}}{3}$
$c^2 = a^2 + b^2 - 2ab\cos C$
余弦定理
在任意三角形中，任意一边的平方等于其他两边的平方和减去这两边与其夹角余弦的二倍积。勾股定理是其特殊情况（C=90°时）。
例题：
在△ABC中，已知a=3，b=4，C=60°，求边c。
解：$c^2 = 3^2 + 4^2 - 2×3×4×\cos 60°$
$c^2 = 9 + 16 - 24×\frac{1}{2} = 25 - 12 = 13$
所以$c = \sqrt{13}$
