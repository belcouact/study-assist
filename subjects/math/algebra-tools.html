🎓 方程基础知识

方程是数学中表示两个表达式相等的等式。掌握方程求解是代数学习的重要基础。

一次方程

形如 ax + b = 0 的方程，其中 a ≠ 0。解法：移项、合并同类项、系数化为1。

二次方程

形如 ax² + bx + c = 0 的方程。可用因式分解、配方法或求根公式求解。

根式方程

含有根号的方程。通常通过平方消除根号，注意检验增根。

分式方程

含有分式的方程。通过通分或去分母转化为整式方程求解。

🏋️ 练习题

1. 解方程：3x - 7 = 2x + 5
解：
3x - 7 = 2x + 5
3x - 2x = 5 + 7
x = 12
2. 解方程：x² - 5x + 6 = 0
解：
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
x = 2 或 x = 3
3. 解方程：√(x + 3) = 5
解：
√(x + 3) = 5
x + 3 = 25
x = 22
检验：√(22 + 3) = √25 = 5 ✓

方程：x/60 + x/40 = 5
