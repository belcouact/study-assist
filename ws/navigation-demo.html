导航组件使用说明

这个页面展示了如何在任何故障相关页面中使用可重用的导航组件。

使用方法

要在任何页面中使用导航组件，只需在页面的 <head> 部分添加以下脚本引用：

<script src="js/navigation-loader.js"></script>

注意：确保脚本引用放在所有其他脚本之后，但在页面内容之前，这样导航栏会首先加载。

功能特点

  • 响应式设计，自动适应PC和移动设备
  • 自动高亮当前页面的导航链接
  • 基于登录状态显示/隐藏退出登录按钮
  • 包含信息图标和模态窗口
  • 移动端菜单切换功能

导航链接

导航组件包含以下链接：

  • 首页 (index.html)
  • 故障列表 (fault-list.html)
  • 新建故障 (new-fault.html)
  • 故障统计 (fault-statistics.html)
  • 信息管理 (info-management.html)
  • 用户资料 (user-profile.html)
  • 意见反馈 (feedback.html)
  • 退出登录 (仅当用户登录时显示)

提示：导航组件会自动检测当前页面路径，并将对应的导航链接设置为活动状态（高亮显示）。

自定义样式

如果需要自定义导航栏的样式，可以通过以下CSS类进行修改：

  • .ws-navigation - 导航栏容器
  • .nav-container - 导航内容容器
  • .page-title - 页面标题
  • .nav-menu - 导航菜单
  • .nav-menu a - 导航链接
  • .nav-menu a.active - 活动导航链接
  • .mobile-menu-toggle - 移动端菜单切换按钮

示例页面

以下是一个使用导航组件的简单页面示例：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>故障列表</title>
    <link rel="stylesheet" href="../css/common.css">
    <link rel="stylesheet" href="../css/responsive.css">
    <script src="js/navigation-loader.js"></script>
</head>
<body>
    <div class="container">
        <h1>故障列表</h1>
        <!-- 页面内容 -->
    </div>
    
    <script>
        // 页面特定的JavaScript代码
    </script>
</body>
</html>

通过这种方式，您可以在任何页面中轻松包含导航功能，而无需重复编写导航代码。
